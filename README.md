# Be My Valentine

A cute and playful interactive Valentine webpage built using HTML, CSS, and JavaScript â€” all in a single file.

The page asks a simple question:

"Will You Be My Valentine?"

## Features

- Soft pink gradient background with a glass-style centered card  
- Cute bunny GIF for visual appeal  
- A YES button that grows larger when hovered  
- A No button that runs away when the user tries to hover over it  
- Floating animated effects when interacting with the YES button  
- Full-screen celebration screen when YES is clicked  

## Built With

- HTML  
- CSS (for styling and animations)  
- Vanilla JavaScript (for interactivity)  

## Project Purpose

This project was created as a fun and creative way to ask someone to be your Valentine while demonstrating interactive front-end development skills using pure HTML, CSS, and JavaScript.
